# TeGe Signature Assets

Static GitHub Pages package for TeGe Outlook signature images.

## Files

```text
index.html
signatures/
  tege-logo.png
  linkedin.png
  booking.png
  business-innovator.png
```

## GitHub Pages URL

```text
https://jorgecastelan-svg.github.io/tege-signature-assets/signatures/
```

Generate production signatures with:

```powershell
pwsh ./scripts/generate-signatures.ps1 -AssetMode url -AssetBaseUrl "https://jorgecastelan-svg.github.io/tege-signature-assets/signatures/"
```
